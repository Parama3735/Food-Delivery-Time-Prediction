# Food Delivery Time Prediction using Machine Learning

## Overview

This project predicts food delivery time using Machine Learning techniques. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation to estimate delivery times accurately.

## Problem Statement

Food delivery services need accurate delivery time estimation to improve customer satisfaction and operational efficiency. This project builds a predictive model using historical delivery data.

## Dataset

The dataset contains information related to food deliveries, including delivery distance, weather conditions, traffic, vehicle type, and delivery time.

* Linear Regression
* Decision Tree Regressor

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

## Results

The trained model predicts food delivery time based on delivery-related features and compares multiple regression algorithms to identify the best-performing model.
